title: グループ管理

---

import { Callout } from 'fumadocs-ui/components/callout';

グループはユーザーごとのチャンネルアクセス権限と課金倍率を分離します。異なるグループのユーザーは、そのグループに割り当てられたチャンネルのみ使用できます。管理者アカウントでログイン後、システム設定または管理パネルで「グループ」の入口を見つけてください。

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：グループ一覧</div>

グループ一覧には設定済みの全グループが表示されます。ユーザーとトークンの両方に特定のグループを割り当てることができます。

<Callout type="info">
  トークンのグループを `auto` に設定すると、システムが優先度順に利用可能なグループを自動的に選択します。クロスグループフェイルオーバーが必要なシナリオに適しています。
</Callout>
