左側ナビゲーションの「ドキュメント」ボタンと「概要」ページは、Root がシステム設定でカスタマイズできます。Root アカウントでログイン後、`/console/setting` にアクセスしてください。

## ドキュメントリンクの設定

1. システム設定ページで「運営設定」タブをクリックします
2. 「ドキュメント URL」入力フィールドを見つけます

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：運営設定タブ — ドキュメント URL 入力フィールド</div>

3. ドキュメントサイトの完全な URL を入力します（例：`https://docs.example.com`）
4. 「保存」をクリックします
5. ホームページに戻ると、左側ナビゲーションバーに「ドキュメント」ボタンが表示され、入力した URL にリンクされます

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：ホームページ左側ナビゲーションに「ドキュメント」ボタンが表示された状態</div>

<Callout type="info">
  ドキュメント URL が空の場合、左側ナビゲーションに「ドキュメント」ボタンは表示されません。
</Callout>

## 概要ページの設定

1. システム設定ページで「その他の設定」タブをクリックします
2. 概要コンテンツ編集エリアを見つけます

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：その他の設定タブ — 概要コンテンツ編集エリア</div>

3. テキストボックスに Markdown 形式のコンテンツを入力します（見出し、リンク、画像、リストなどに対応）
4. 「保存」をクリックします
5. ユーザーが左側ナビゲーションの「概要」をクリックすると、設定したコンテンツが表示されます

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：概要ページの表示結果</div>
