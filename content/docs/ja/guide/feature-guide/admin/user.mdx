---
title: ユーザー管理
description: プラットフォームの全登録ユーザーを確認・管理
---

プラットフォームの全登録ユーザーを確認・管理します。ロール、クォータ、グループの変更やアカウントステータスの制御が可能です。管理者アカウントでログイン後、左側ナビゲーションの「ユーザー」をクリックするか、`/console/user` に直接アクセスしてください。

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：ユーザー一覧ページ</div>

ユーザー一覧には全登録ユーザーのユーザー名、メール、ロール、グループ、クォータ残高、ステータスが表示されます。

## ユーザーの編集

1. 一覧で対象ユーザーを見つけ、右側の「編集」ボタンをクリックすると編集ダイアログが表示されます

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：ユーザー編集ダイアログ</div>

2. 以下の情報を変更できます：
   - **ロール**：一般ユーザー / 管理者 — 変更は即座に反映されます
   - **グループ**：ユーザーをチャンネルグループに割り当てます
   - **クォータ残高**：ユーザーのクォータ値を直接設定します
   - **ステータス**：アカウントを有効化または無効化します。無効化されたユーザーはログインできません
3. 完了したら「保存」をクリックします

## ユーザーの検索

1. ユーザー一覧上部の検索ボックスを見つけます
2. ユーザー名またはメールのキーワードを入力します

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：キーワード入力後のフィルタリング結果</div>

3. 一覧がリアルタイムでフィルタリングされ、一致するユーザーレコードが表示されます
