---
title: ログと統計
description: プラットフォーム全体の API 呼び出し記録と消費統計を確認
---

プラットフォーム全体の API 呼び出し記録と消費統計を確認できます。ユーザー、モデル、チャンネルなど多次元での分析に対応しています。管理者アカウントでログイン後、左側ナビゲーションの「ログ」をクリックするか、`/console/log` に直接アクセスしてください。

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：プラットフォーム全体ログ一覧</div>

管理者はすべてのユーザーの API 呼び出し記録を確認できます。一般ユーザーには表示されないユーザー名やチャンネル名の列が含まれます。

## ログ統計パネル

ログページ上部の統計サマリーエリアを確認してください。

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：ログ統計パネル</div>

統計パネルにはプラットフォーム全体の呼び出し量の合計が表示され、管理者が全体的な使用状況を把握するのに役立ちます。

## プラットフォーム消費トレンド

左側ナビゲーションの「データダッシュボード」をクリックするか、`/console` に直接アクセスしてください（管理者ビュー）。

<div style={{background:'#f3f4f6',borderRadius:'8px',padding:'48px 24px',textAlign:'center',color:'#9ca3af',marginBottom:'16px',border:'1px dashed #d1d5db'}}>📷 截图占位：プラットフォーム消費トレンドグラフ</div>

管理者ダッシュボードにはプラットフォーム全体の消費トレンドとユーザー別消費分布が表示されます。
