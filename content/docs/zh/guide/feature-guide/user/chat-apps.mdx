TokenJump 支持快速导入配置到多种 AI 聊天应用，方便测试和日常使用。

## 一键导入配置

在控制台→令牌页面可以一键导入配置到支持的聊天应用中。

![一键导入配置](/assets/guide/import-chat-config.png)

### 使用方法

1. 在令牌列表页找到要使用的令牌
2. 点击令牌右侧的「导入」或「配置」按钮
3. 选择目标聊天应用
4. 系统自动跳转到对应应用并填充配置信息
